# AI Text Summarizer

This project is a **text summarizer** built using **Hugging Face Transformers**.  
It can summarize any paragraph, article, or text you provide.

## How to Use

1. Open the `summarizer.ipynb` notebook in Jupyter Notebook or VS Code.  
2. Replace the `text` variable with your paragraph or article.  
3. Run the notebook to get a concise summary.

## Technologies Used

- Python
- Jupyter Notebook
- Hugging Face Transformers (`sshleifer/distilbart-cnn-12-6`)
